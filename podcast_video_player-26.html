Numbered files (01, 02, 03, etc.): Upload files to your hosting location (e.g., same GitHub Release as your loop video).
Regular keys 1-8: Play videos/images (formats: .mp4, .mov, .jpg, .png, .gif, .avi, .webm)
Numpad keys 1-8: Play sounds over current media (formats: .mp3, .wav, .m4a)
Sound file naming (GitHub Releases): Use any descriptive name starting with the number (e.g., 01-sad-trombone.m4a, 02-applause.mp3)
Sound file naming (other hosting): Use simple patterns: 01.m4a, 01-sound.m4a, 01-audio.wav, etc.
